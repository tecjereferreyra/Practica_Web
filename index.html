C#

Es un lenguaje de programación moderno, orientado a objetos y con seguridad de tipos, desarrollado por Microsoft y parte de la plataforma .NET. Es conocido por ser multiplataforma, eficiente y fácil de aprender, ideal para crear aplicaciones web, de escritorio, móviles y juegos.

Galería de Lenguajes

Java Script

JavaScript (JS) es un lenguaje de programación ligero, interpretado y basado en prototipos, esencial para crear sitios web interactivos y dinámicos. Es el estándar para el desarrollo frontend en navegadores, aunque también funciona en el backend con Node.js. Permite animaciones, validación de formularios y manejo de datos en tiempo real.

Java

Java es un lenguaje de programación de alto nivel, orientado a objetos y una plataforma informática, lanzado en 1995 por Sun Microsystems (ahora propiedad de Oracle). Es conocido por ser multiplataforma ("Write once, run anywhere"), lo que permite que el código se ejecute en cualquier dispositivo con la Java Virtual Machine (JVM). Se usa ampliamente en aplicaciones empresariales, Android, web y datos.

Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, reconocido por su sintaxis sencilla y legible, similar al inglés. Es muy popular por su versatilidad, código abierto (gratuito) y su amplio uso en inteligencia artificial, ciencia de datos, desarrollo web (backend) y automatización.
